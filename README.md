# Getting Started
Like my previous projects, this one doesn't have prebuilt binaries. So, you will need to build it yourself (sorry).

## Prerequisites
In order to build this project, you are going to need:
* [Meson](https://mesonbuild.com/index.html)

## Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/dat-alpaca/YAGNI-dates
    cd YAGNI-dates
    ```
    <br>

2. **Set up the project**
    This project doesn't have any build options. All you need to do is run it.
    
    ```bash
    ./run german
    ```

    The command line argument is the language you want to use.

    This will create a `deploy` folder with the default languages and an executable called `yagni`.
